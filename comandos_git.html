COMPROBAR LA INSTALACIO DE GIT: git --version

CONFIGURACION DE GIT

NOMBRE DEL ADMINISTRADOR: git config --global user.name "p72losaj"

COMPROBAR LA INSTALACION: gitconfig--global user.email p72losaj@uco.es

EDITOR DE TEXTO: gitconfig--global core.editor “gedit”

HABILITAR COLOR DE LA INTERFAZ: gitconfig--global color.ui true

LISTAR LA CONFIGURACION: gitconfig --list

Almacenar los datos de acceso a github y no tener que volver a utilizarlo: git remote set-url origin git@github.com:p72losaj/IS-2122.git

USO BASICO DE GIT

CREAR UN REPOSITORIO:
Crear la carpeta del proyecto: mkdir IS-2122
Iniciar repositorio en un directorio: git init IS-2122
Acceder al directorio del proyecto: cd IS-2122

Añadir ficheros al area de staging o preparacion:
Anadir un fichero concreto: git add fichero.txt
Anadir todos los ficheros posibles: git add -A

Confirmar los cambios: git commit -m "Mensaje descriptivo"

Ver estado o cambios del directorio: git STATUS

Ver diferencia entre ficheros en el directorio y en el repositorio git:
get diff
get diff --color-words

Ver diferencia entre ficheros preparados y el repositorio: git diff --staged

Historial de commits: git log

Dar formato al log: git config --global alias.hist

Anadir etiqueta en el commit actual: git tag etiqueta

Ignorar ficheros o directorios en el repositorio: fichero .gitignore

Retirar archivos del stagging: git reset HEAD archivo

Complementar/Modificar último commit: git commit --amend -m "Mensaje"

Revertir ultimo commit: git revert HEAD

Revertir a un commit especifico:
git reset --soft id_commit
git reset --mixed id_commit
Volver al commit id_comit: git reset --hard id_commit

Eliminar archivos
git rm archivo
git rm --cached archivo
git commit -m "Mensaje"

Mover o renombrar archivos
git mv archivoAntiguo ArchivoNuevo
git commit -m "Mensaje"

Listar el contenido del repositorio: git ls-tree master

Examinar el contenido de un commit: git shown id

Comparar un commit con el actual: git diff id fichero

GESTION DE RAMAS

Ver listado de ramas: git branch

Crear una rama: git branch nombre_rama

Cambiarse a una rama: git checkout nombre_rama

Integrar una rama en la actual: git merge nombre_rama

Comparar ramas: git diff nombre_rama1 ... nombre_ramaN

Renombrar ramas: git branch -m nombre_antiguo nombre_nuevo

Eliminar ramas:
git branch -d nombre_rama
git branch -D nombre_rama

Crear/clonar repositorio

Clonar repositorio remoto: git clone git@github.com:p72losaj/IS-2122.git
