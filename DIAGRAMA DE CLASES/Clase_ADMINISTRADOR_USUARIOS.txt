Clase que gestiona las funcionalidades sobre los usuarios. Hereda la clase administrador

OPERACIONES:

+ ADMINISTRADOR_USUARIOS(): Constructor de clase

* Parametro nombre: String -> Nombre del usuario

* Parametro email: String -> Email del usuario

* Parametro contrasena: String -> Contrasena del usuario

* Parametro dni: String -> dni del usuario

* Parametro tipo: String -> Tipo del usuario

* Parametro nucleos: int -> Nucleos reservables del usuario

* Parametro limite: int -> Limite de tiempo de reservas del usuario
